
#  SQL Database Project

This repository contains a complete SQL script for building a fully structured relational database. The project covers table creation, relationships, constraints, indexing, and large-scale data insertion. 


##  Project Description

This SQL project demonstrates how to:

- Create a database from scratch  
- Build relational tables  
- Define primary and foreign keys  
- Create indexes for performance optimization  
- Insert realistic sample data  
- Prepare a ready-to-query database for analysis  

##  Database Structure

The script creates five major tables:

- **Customer** – Customer details and contact information  
- **Supplier** – Supplier company and contact details  
- **Product** – Product information with supplier linkage  
- **Order** – Customer order records  
- **OrderItem** – Line-item details for each order  

Each table is fully linked through foreign key constraints.

##  Key Features

- Complete relational schema  
- Foreign key relationships  
- Indexes to improve query performance  
- Hundreds of populated rows  
- Realistic retail-focused dataset  
- Useful for analytics, SQL practice, and BI dashboards
